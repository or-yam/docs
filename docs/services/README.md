# Greenpress Services

architectural design for all services

## Default Ports

By default, greenpress services are running on those ports:

- `3000` Blog Front
- `3001` Admin Panel
- `9000` Authentication service 
- `9001` Content service
- `9002` Secrets service 
- `9003` Assets service
- `9004` Mailing service
- `9005` Drafts service
